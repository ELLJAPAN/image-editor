# EC画像加工ツール (eBay対応)

画像のAI背景除去・トリミング・リサイズ・画質調整を一括処理するWebツール。

## 機能
- 🤖 AI背景除去（rembg）
- ✂️ トリミング
- 📏 リサイズ（複数モード）
- 🎨 明るさ・コントラスト・鮮やかさ調整
- 🛍️ eBayモード（1600px・正方形補正）
- 📦 一括処理・ZIPダウンロード
